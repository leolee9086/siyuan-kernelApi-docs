本文档提供思源笔记的API接口说明，可用于开发第三方应用或插件。

API基础URL: http://127.0.0.1:6806

请求方法：除非特别说明，均使用 POST 方法

认证方式：在 设置 - 关于 中查看 API token，在请求标头中添加 Authorization: Token xxx

简介

思源笔记是一款本地优先的个人知识管理系统，支持完全离线使用，同时也支持端到端加密同步。

通过API，你可以：

  • 创建、修改、删除笔记和笔记本
  • 插入、更新和删除内容块
  • 查询数据库内容
  • 导出各种格式的文档
  • 设置和获取块属性
  • 管理文件和资源
  • 更多其他功能...

开始使用

选择左侧的API类别和具体接口，查看详细的API文档。每个API页面包含：

  • 接口描述
  • 请求参数
  • 返回值
  • 示例代码
  • 在线测试功能（需要开启思源笔记才能使用）

认证

// 在请求头中添加认证信息
fetch('http://127.0.0.1:6806/api/system/currentTime', {
    method: 'POST',
    headers: {
        'Authorization': 'Token your-api-token',
        'Content-Type': 'application/json'
    }
})
.then(response => response.json())
.then(data => console.log(data));
