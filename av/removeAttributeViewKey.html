POST /api/av/removeAttributeViewKey

移除属性视图列 (Key)

源文件 (GitHub)
需要认证

接口描述

从指定的属性视图（数据库）中移除列 (Key)。

请求参数

参数名 类型 必选 描述
avID string 是 属性视图 ID。
keyID string 是 要移除的列 (Key) 的 ID。

返回值

操作成功时不返回 `data`。

参数名 类型 描述
codenumber返回码，0表示成功
msgstring返回信息
datanull成功时为 null。
