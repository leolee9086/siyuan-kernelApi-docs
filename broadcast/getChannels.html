获取所有广播频道

在 GitHub 上查看源码

POST /api/broadcast/getChannels

描述

获取当前存在的所有广播频道及其信息（名称和连接数）。

请求参数

此接口不需要请求参数。

响应结果

响应体是 JSON 格式。code 为 0 表示成功。data 包含一个频道信息对象的数组。


{
  "code": 0,
  "msg": "",
  "data": {
    "channels": [
      {
        "name": "channel-1",
        "count": 3
      },
      {
        "name": "channel-2",
        "count": 1
      }
      // ... more channels
    ]
  }
}
