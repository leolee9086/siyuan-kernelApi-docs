POST /api/filetree/setDocAttrs

设置文档属性

需要认证

接口描述

设置指定文档的属性。可以设置文档的图标、备注等。此接口用于修改文档本身的属性，而非块属性。

请求参数

参数名 类型 必选 描述
id string 是 文档ID
attrs object 是 要设置的属性键值对
attrs.icon string 否 文档图标 (Emoji Unicode 或 资源名，如"1f4dd" 或 "assets/icon.png")
attrs.memo string 否 文档备注
attrs.name string 否 文档别名
attrs.alias string 否 文档别名（同name）
attrs.bookmark string 否 文档书签

返回值

参数名 类型 描述
code number 返回码，0表示成功
msg string 返回信息
data null 无返回数据

注意： 设置属性时，如果某个属性值为空字符串（""），则会移除该属性。

返回示例

请求示例

POST /api/filetree/setDocAttrs HTTP/1.1
Host: 127.0.0.1:6806
Content-Type: application/json
Authorization: Token your-token

{
  "id": "20220301153724-vy37rik",
  "attrs": {
    "icon": "1f680",
    "memo": "这是一条新的文档备注",
    "name": "SiyuanIntro"
  }
}

返回示例

{
  "code": 0,
  "msg": "",
  "data": null
}
