仓库/版本库 API (Repo)

首页

与思源笔记内置版本库（快照）相关的 API。

API 列表

  • /api/repo/createSnapshot - 创建版本库快照
  • /api/repo/checkoutRepo - 检出版本库快照
  • /api/repo/diffRepoSnapshots - 对比两个快照的差异
  • /api/repo/downloadCloudSnapshot - 从云端下载快照
  • /api/repo/getCloudRepoSnapshots - 获取云端仓库的快照列表
  • /api/repo/getCloudRepoTagSnapshots - 获取云端仓库指定标签的快照列表
  • /api/repo/getRepoFile - 获取快照中的文件内容
  • /api/repo/getRepoSnapshots - 获取本地仓库的快照列表（分页）
  • /api/repo/getRepoTagSnapshots - 获取本地仓库指定标签的快照列表
  • /api/repo/importRepoKey - 导入仓库密钥
  • /api/repo/initRepoKey - 初始化仓库密钥
  • /api/repo/initRepoKeyFromPassphrase - 从口令初始化仓库密钥
  • /api/repo/openRepoSnapshotDoc - 打开快照中的文档内容
  • /api/repo/purgeCloudRepo - 清理云端仓库
  • /api/repo/purgeRepo - 清理本地仓库
  • /api/repo/removeCloudRepoTagSnapshot - 删除云端仓库指定标签的快照
  • /api/repo/removeRepoTagSnapshot - 删除本地仓库指定标签的快照
  • /api/repo/resetRepo - 重置仓库
  • /api/repo/tagSnapshot - 为快照打标签
  • /api/repo/uploadCloudSnapshot - 上传快照到云端
