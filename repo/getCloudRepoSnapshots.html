获取云端快照列表 (getCloudRepoSnapshots)

首页 | Repo API | GitHub Source

功能描述

此 API 端点 POST /api/repo/getCloudRepoSnapshots 用于获取已配置的云端存储中的版本库快照列表。

返回的结果支持分页。

请求参数

请求体必须是 JSON 格式，包含以下参数：

  • page (number): 必填。要获取的快照列表的页码，从 1 开始计数。

请求示例 (获取第 1 页):


{
  "page": 1
}

响应结果

成功的请求将返回一个 JSON 对象，code 为 0，data 字段包含快照列表和分页信息。


{
  "code": 0,
  "msg": "",
  "data": {
    "snapshots": [
      {
        "time": 1666857600,
        "hTime": "2023-10-27 10:00:00",
        "id": "20231027100000-hijklmn",
        "size": 103424,
        "hSize": "101 KB",
        "tag": "v1.0" // 快照标签
      },
      {
        "time": 1666771200,
        "hTime": "2023-10-26 10:00:00",
        "id": "20231026100000-abcdefg",
        "size": 102400,
        "hSize": "100 KB",
        "tag": ""
      }
      // ... 更多快照信息 ...
    ],
    "pageCount": 5, // 总页数
    "totalCount": 48 // 快照总数
  }
}
  • code: 0 表示成功，非 0 表示失败（例如连接云端失败、认证失败等）。
  • msg: 错误信息（如果 code 非 0）。
  • data.snapshots: 一个数组，包含当前页的快照信息对象。每个对象包含 `id`, `time` (时间戳), `hTime` (可读时间), `size` (字节大小), `hSize` (可读大小), `tag` (标签名) 等字段。
  • data.pageCount: 云端快照的总页数。
  • data.totalCount: 云端快照的总数量。
