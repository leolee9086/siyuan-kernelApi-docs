POST /api/ai/chatGPT

基础 AI 对话

源文件 (GitHub)
需要认证 需要管理员

接口描述

向配置好的 AI 服务（目前主要指 OpenAI ChatGPT 或兼容接口）发送文本消息，并获取 AI 的回复。需要先在设置中配置好 AI 服务。

请求参数

参数名 类型 必选 描述
msg string 是 要发送给 AI 的文本消息。

返回值

返回 AI 的回复内容。

参数名 类型 描述
codenumber返回码，0表示成功
msgstring返回信息
datastringAI 回复的文本内容。如果 AI 服务出错或未配置，可能返回空字符串或错误信息。
 
{
  "msg": "你好，请介绍一下你自己。"
}
{
  "code": 0,
  "msg": "",
  "data": "你好！我是一个大型语言模型，由 Google 训练。"
}

错误示例 (例如未配置 AI):

{
  "code": -1,
  "msg": "AI provider has not been set",
  "data": ""
}

注意：这是一个社区维护的文档，可能与官方最新版本存在差异。
