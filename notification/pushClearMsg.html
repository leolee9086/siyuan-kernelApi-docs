清除消息

在 GitHub 上查看源码

POST /api/notification/pushClearMsg

描述

清除先前通过 /api/notification/pushMsg 或 /api/notification/pushErrMsg 推送并仍在显示的指定通知消息。需要提供消息的唯一 ID 来指定要清除哪条消息。

请求参数

请求体需要是 JSON 格式。

参数名 类型 描述 必需
id string 要清除的消息的唯一标识符，由 pushMsg 或 pushErrMsg 接口返回 是

响应结果

响应体是 JSON 格式。code 为 0 表示成功，即使提供的 ID 不存在或消息已消失，接口也会返回成功。


{
  "code": 0,
  "msg": "",
  "data": null
}

补充说明

此 API 通过 WebSocket 向所有连接的界面广播清除消息的指令。如果消息 ID 对应的通知当前正在显示，则会被移除。
