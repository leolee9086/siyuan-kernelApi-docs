POST /api/notebook/lsNotebooks

列出笔记本

需要认证

接口描述

获取所有可用笔记本的列表，包括笔记本ID、名称、图标、排序和关闭状态等信息。

请求参数

此接口不需要参数。

返回值

参数名 类型 描述
code number 返回码，0 表示成功
msg string 错误信息，成功时为空字符串
data.notebooks array 笔记本列表
data.notebooks[].id string 笔记本ID
data.notebooks[].name string 笔记本名称
data.notebooks[].icon string 笔记本图标（Unicode 字符）
data.notebooks[].sort number 笔记本排序顺序
data.notebooks[].closed boolean 笔记本是否处于关闭状态
请求示例
返回示例

Host: 127.0.0.1:6806
Content-Type: application/json
Authorization: Token your-api-token
