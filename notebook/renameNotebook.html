POST /api/notebook/renameNotebook

重命名笔记本

需要认证

接口描述

修改现有笔记本的名称。

请求参数

参数名 类型 必选 描述
notebook string 是 笔记本ID
name string 是 笔记本的新名称

返回值

参数名 类型 描述
code number 返回码，0 表示成功
msg string 错误信息，成功时为空字符串
data null 重命名成功时为 null
请求示例
返回示例

请求示例

POST /api/notebook/renameNotebook HTTP/1.1
Host: 127.0.0.1:6806
Content-Type: application/json
Authorization: Token your-api-token

{
  "notebook": "20210831090520-7dvbdv0",
  "name": "笔记本的新名称"
}

返回示例

{
  "code": 0,
  "msg": "",
  "data": null
}
