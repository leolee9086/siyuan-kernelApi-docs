接口描述

解压一个指定的压缩文件到指定目录。

请求参数

（待补充详细参数）

参数名 类型 必选 描述
path string 是 要解压的压缩文件路径
dest string 是 解压目标目录路径

返回值

标准返回格式，data 为 null。

参数名 类型 描述
codenumber返回码，0表示成功
msgstring返回信息
datanull固定为 null

返回示例

请求示例

{
    "path": "assets/archive.zip",
    "dest": "temp/unzipped"
}

返回示例

{
    "code": 0,
    "msg": "",
    "data": null
}
