API/WS 列表

  • /api/bazaar/batchUpdatePackage - 批量更新指定前端的集市包信息（例如，从远程仓库拉取最新的包列表，检查已安装包的更新等）。这是一个后台操作，API 调用本身只负责触发任务。
  • /api/bazaar/getBazaarIcon - 获取集市中可用的图标包列表。
  • /api/bazaar/getBazaarPackageREAME - 获取指定集市包（如插件、主题、图标包等）的 README 文件内容。思源笔记会将其转换为 HTML 格式返回，方便在界面上直接展示包的详细说明。
  • /api/bazaar/getBazaarPlugin - 从官方集市获取可用的插件列表。可以通过关键词进行筛选。
  • /api/bazaar/getBazaarTemplate - 获取集市中可用的模板包列表。模板包就是你在思源笔记中新建文档时可以选择套用的那些模板。
  • /api/bazaar/getBazaarTheme - 获取集市中可用的主题包列表。主题决定了思源笔记的整体外观风格。
  • /api/bazaar/getBazaarWidget - 获取集市中可用的挂件包 (Widget) 列表。挂件是可以嵌入到文档中的小应用或信息展示块，例如时钟、日历、天气等。
  • /api/bazaar/getInstalledIcon - 获取当前工作空间中**已安装**的图标包列表。
  • /api/bazaar/getInstalledPlugin - 获取当前工作空间中已安装的插件列表。可以通过关键词进行筛选。
  • /api/bazaar/getInstalledTemplate - 获取当前工作空间中**已安装**的模板包列表。
  • /api/bazaar/getInstalledTheme - 获取当前工作空间中**已安装**的主题包列表。
  • /api/bazaar/getInstalledWidget - 获取当前工作空间中**已安装**的挂件包列表。
  • /api/bazaar/getUpdatedPackage - 获取所有已安装的集市包（插件、挂件、图标、主题、模板）中需要更新的列表。
  • /api/bazaar/installBazaarIcon - 从集市下载并安装指定的图标包。
  • /api/bazaar/installBazaarPlugin - 从集市下载并安装指定的插件。
  • /api/bazaar/installBazaarTemplate - 从集市下载并安装指定的模板。
  • /api/bazaar/installBazaarTheme - 从集市下载并安装指定的主题。
