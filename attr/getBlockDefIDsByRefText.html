POST /api/attr/getBlockDefIDsByRefText

通过引用文本获取块定义 ID

源文件 (GitHub)
需要认证

接口描述

根据给定的引用文本（例如 `((块ID "引用文本"))` 中的 "引用文本" 部分），查找并返回包含该引用文本的所有定义块（通常是命名锚点或标题）的 ID。

请求参数

参数名 类型 必选 描述
refText string 是 要查找的引用文本。
excludeIDs string[] 否 要排除在搜索结果之外的块 ID 列表。

返回值

返回包含匹配引用文本的定义块 ID 列表。

参数名 类型 描述
codenumber返回码，0表示成功
msgstring返回信息
datastring[]包含匹配引用文本的定义块 ID 列表。
请求示例 (cURL)
返回示例

请求示例 (cURL)

curl -X POST \
  http://127.0.0.1:6806/api/attr/getBlockDefIDsByRefText \
  -H 'Authorization: Token YOUR_API_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "refText": "我的定义块",
    "excludeIDs": ["20231027150000-excluded1"]
}'

返回示例

{
    "code": 0,
    "msg": "",
    "data": [
        "20231027150000-defblock1",
        "20231027150000-defblock2"
    ]
}
