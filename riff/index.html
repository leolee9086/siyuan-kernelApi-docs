Riff (闪卡) API

首页

这里包含了与思源笔记闪卡 (Riff Card) 功能相关的 API。

API 列表

  • /api/riff/addRiffCards - 添加闪卡
  • /api/riff/batchSetRiffCardsDueTime - 批量设置闪卡到期时间
  • /api/riff/createRiffDeck - 创建闪卡包
  • /api/riff/getNotebookRiffCards - 获取笔记本内闪卡列表
  • /api/riff/getNotebookRiffDueCards - 获取笔记本内到期闪卡列表
  • /api/riff/getRiffCards - 获取闪卡包内闪卡列表
  • /api/riff/getRiffCardsByBlockIDs - 根据块ID获取闪卡信息
  • /api/riff/getRiffDueCards - 获取闪卡包内到期闪卡列表
  • /api/riff/getTreeRiffCards - 获取文档树内闪卡列表
  • /api/riff/getTreeRiffDueCards - 获取文档树内到期闪卡列表
  • /api/riff/removeRiffCards - 移除闪卡
  • /api/riff/removeRiffDeck - 删除闪卡包 (高风险!)
  • /api/riff/renameRiffDeck - 重命名闪卡包
  • /api/riff/resetRiffCards - 重置闪卡复习状态
  • /api/riff/reviewRiffCard - 复习闪卡
  • /api/riff/skipReviewRiffCard - 跳过闪卡复习
