Get Export Progress

POST /api/export/getExportProgress

获取导出进度。

请求参数

此接口无需请求参数。

返回数据

当前的导出进度。


{
    "code": 0,
    "msg": "",
    "data": {
        "percent": 0 // 导出进度百分比
    }
}

请求示例


curl -X POST \
  http://127.0.0.1:6806/api/export/getExportProgress \
  -H 'Authorization: Token YOUR_API_TOKEN' \
  -H 'Content-Type: application/json'

响应示例


{
  "code": 0,
  "msg": "",
  "data": {
    "percent": 50 // 当前导出进度为 50%
  }
}
