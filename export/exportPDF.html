POST /api/export/exportPDF

导出文档为PDF

需要认证

接口描述

将指定的文档（包含其所有块）导出为PDF文件。支持多种导出配置选项。

请求参数

参数名 类型 必选 描述
id string 是 要导出的文档ID
pdf object 否 PDF导出配置选项 (详细见下方)
pdf.landscape boolean 否 是否横向页面，默认为 false (纵向)
pdf.marginType number 否 边距类型，0: 默认, 1: 无, 2: 最小。默认为 0
pdf.pageSize string 否 页面尺寸，如 'A4', 'A3', 'Letter' 等。默认为 'A4'
pdf.scale number 否 缩放比例，范围 0.1 - 2。默认为 1

返回值

参数名 类型 描述
code number 返回码，0表示成功
msg string 返回信息
data object 返回数据
data.hPath string 导出的PDF文件的人类可读路径（相对于工作空间导出目录）
data.path string 导出的PDF文件的绝对路径

注意： 导出的PDF文件将保存在工作空间的 `export` 目录下。

请求示例
返回示例

请求示例 (基本)

POST /api/export/exportPDF HTTP/1.1
Host: 127.0.0.1:6806
Content-Type: application/json
Authorization: Token your-token

{
  "id": "20220301153724-vy37rik"
}

请求示例 (带配置)

POST /api/export/exportPDF HTTP/1.1
Host: 127.0.0.1:6806
Content-Type: application/json
Authorization: Token your-token

{
  "id": "20220301153724-vy37rik",
  "pdf": {
    "landscape": true,
    "marginType": 2,
    "pageSize": "A3",
    "scale": 0.8
  }
}

返回示例

{
  "code": 0,
  "msg": "",
  "data": {
    "hPath": "/export/思源笔记简介-20240417233000.pdf",
    "path": "D:/SiYuan/data/export/思源笔记简介-20240417233000.pdf"
  }
}
