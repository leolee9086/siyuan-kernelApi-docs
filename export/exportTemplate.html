Export Template

POST /api/export/exportTemplate

导出模板。

请求参数

参数名 类型 是否必需 描述
id string 是 要导出的模板块 ID。

返回数据

成功导出的模板文件路径。


{
    "code": 0,
    "msg": "",
    "data": {
        "path": "string" // 导出的模板文件路径
    }
}

请求示例


curl -X POST \
  http://127.0.0.1:6806/api/export/exportTemplate \
  -H 'Authorization: Token YOUR_API_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "id": "20230101..." // 模板块 ID
  }'

响应示例


{
  "code": 0,
  "msg": "",
  "data": {
    "path": "/path/to/exported/template.sy"
  }
}
