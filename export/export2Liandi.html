导出到链滴

在 GitHub 上查看源码

POST /api/export/export2Liandi

描述

将指定的文档块（及其内容）导出并发布到链滴社区（需要用户已登录链滴账号）。

请求参数

请求体需要是 JSON 格式。

参数名 类型 描述 必需
id string 要导出的文档块 ID。 是

响应结果

响应体是 JSON 格式。code 为 0 表示成功发布，非 0 表示失败。成功时 data 为 null。


{
  "code": 0,
  "msg": "",
  "data": null
}
