导出为 MediaWiki

在 GitHub 上查看源码

POST /api/export/exportMediaWiki

描述

将指定的文档块（ID）通过 Pandoc 转换为 MediaWiki 格式，并打包为 ZIP 文件。需要系统安装 Pandoc。

请求参数

请求体需要是 JSON 格式。

参数名 类型 描述 必需
id string 要导出的文档块 ID。 是

响应结果

响应体是 JSON 格式。code 为 0 表示成功。data 包含导出的 ZIP 文件信息。


{
  "code": 0,
  "msg": "",
  "data": {
    "name": "document_title.wiki", // 导出的文件名 (基于文档标题)
    "zip": "/temp/export/xxx.zip"   // 导出的 ZIP 文件相对路径
  }
}
