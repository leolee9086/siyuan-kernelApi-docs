POST /api/export/exportNotebook

导出笔记本

源文件 (GitHub)
需要认证 需要管理员权限

接口描述

将指定的笔记本导出为标准 Markdown 文件（带资源文件）或 `.sy` 格式文件。

请求参数

参数名 类型 必选 描述
id string 是 要导出的笔记本的 ID。
sy boolean 是 是否导出为 `.sy` 格式。`true` 表示导出 `.sy` 文件，`false` 表示导出标准 Markdown 文件。

返回值

操作成功时不返回 `data`。导出的文件会保存在工作空间的 `export` 目录下。

参数名 类型 描述
codenumber返回码，0表示成功
msgstring返回信息
datanull成功时为 null
请求示例 (cURL)
返回示例

导出为 Markdown (cURL)

curl -X POST \
  http://127.0.0.1:6806/api/export/exportNotebook \
  -H 'Authorization: Token YOUR_API_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "id": "20200812220555-ht00000",
    "sy": false
}'

导出为 .sy (cURL)

curl -X POST \
  http://127.0.0.1:6806/api/export/exportNotebook \
  -H 'Authorization: Token YOUR_API_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "id": "20200812220555-ht00000",
    "sy": true
}'

返回示例

{
    "code": 0,
    "msg": "",
    "data": null
}
