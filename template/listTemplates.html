POST /api/template/listTemplates

获取所有模板片段

需要认证

接口描述

获取工作空间内所有已定义的模板片段列表。返回的列表包含了每个模板的路径和内容摘要。

请求参数

参数名 类型 必选 描述
无需参数

返回值

参数名 类型 描述
code number 返回码，0表示成功
msg string 返回信息
data array 模板片段列表
data[].path string 模板文件路径
data[].content string 模板内容（通常是片段）
请求示例
返回示例

请求示例

POST /api/template/listTemplates HTTP/1.1
Host: 127.0.0.1:6806
Content-Type: application/json
Authorization: Token your-token

{}
