POST /api/template/createTemplate

创建模板片段

需要认证

接口描述

在指定的路径下创建一个新的模板片段。模板内容应为Markdown格式。

请求参数

参数名 类型 必选 描述
path string 是 模板保存的完整路径，包含文件名，如"/data/templates/我的新模板.md"
content string 是 模板的Markdown内容

返回值

参数名 类型 描述
code number 返回码，0表示成功
msg string 返回信息
data null 无返回数据

注意： 路径参数 `path` 必须是相对于工作空间根目录的完整路径，并且包含 `.md` 文件扩展名。

请求示例
返回示例

请求示例

POST /api/template/createTemplate HTTP/1.1
Host: 127.0.0.1:6806
Content-Type: application/json
Authorization: Token your-token

{
  "path": "/data/templates/我的新模板.md",
  "content": "# 我的新模板\n\n这是一个通过API创建的新模板。\n\n变量示例: {{myVariable}}"
}

返回示例

{
  "code": 0,
  "msg": "",
  "data": null
}
