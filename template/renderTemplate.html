POST /api/template/renderTemplate

渲染模板片段

需要认证

接口描述

根据提供的变量渲染指定的模板片段。模板中的变量将使用`vars`参数中的值进行替换。

请求参数

参数名 类型 必选 描述
path string 是 模板文件路径，可以通过`listTemplates`接口获取
vars object 否 用于替换模板中变量的键值对。键为模板中的变量名（不带`{{`和`}}`），值为替换后的内容。

返回值

参数名 类型 描述
code number 返回码，0表示成功
msg string 返回信息
data string 渲染后的Markdown文本内容

模板语法示例

模板中使用双大括号 `{{变量名}}` 来定义变量。例如：

# 会议纪要

**时间**: {{time}}
**地点**: {{location}}
**参会人员**: {{attendees}}

## 议题

- {{topic1}}
- {{topic2}}

使用此接口时，可以通过`vars`参数传入实际的值：

{
  "path": "/data/templates/会议纪要.md",
  "vars": {
    "time": "2024-04-17 10:00",
    "location": "会议室A",
    "attendees": "张三, 李四",
    "topic1": "讨论项目进展",
    "topic2": "确定下一步计划"
  }
}

接口将返回渲染后的Markdown文本。
