POST /api/file/removeDir

删除目录

源文件 (GitHub)
需要认证 需要管理员权限
警告： 删除操作不可恢复，会递归删除目录下的所有内容，请谨慎操作！

接口描述

删除指定路径的目录及其包含的所有文件和子目录。

请求参数

参数名 类型 必选 描述
path string 是 要删除的目录路径，相对路径（相对于工作空间）。例如 `/data/to-be-deleted`。

返回值

操作成功时不返回 `data`。

参数名 类型 描述
codenumber返回码，0表示成功
msgstring返回信息
datanull成功时为 null
请求示例 (cURL)
返回示例

请求示例 (cURL)

curl -X POST \
  http://127.0.0.1:6806/api/file/removeDir \
  -H 'Authorization: Token YOUR_API_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "path": "/data/obsolete-folder" 
}'

返回示例

{
    "code": 0,
    "msg": "",
    "data": null
}
