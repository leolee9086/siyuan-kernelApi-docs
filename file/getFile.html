POST /api/file/getFile

获取文件

需要认证

接口描述

获取工作空间中的文件。该接口可以用于获取思源笔记工作空间内的任何文件内容，包括但不限于笔记、图片、附件等。

请求参数

参数名 类型 必选 描述
path string 是 文件路径，相对于工作空间根目录

返回值

如果请求成功，会直接返回文件内容，Content-Type 会根据文件类型自动设置。

如果请求失败，会返回错误信息，格式如下：

参数名 类型 描述
code number 返回码，-1表示参数错误，403表示禁止访问，404表示文件不存在，500表示服务器错误
msg string 错误信息
请求示例
返回示例

请求示例

POST /api/file/getFile HTTP/1.1
Host: 127.0.0.1:6806
Content-Type: application/json
Authorization: Token your-api-token

{
  "path": "/data/assets/image.png"
}

返回示例

成功时返回文件内容（二进制数据）

失败时返回错误信息：

{
  "code": 404,
  "msg": "stat /data/assets/image.png: no such file or directory"
}
