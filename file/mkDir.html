POST /api/file/mkDir

创建目录

源文件 (GitHub)
需要认证 需要管理员权限

接口描述

在指定路径下创建目录。如果父目录不存在，会自动创建。

请求参数

参数名 类型 必选 描述
path string 是 要创建的目录路径，相对路径（相对于工作空间）。例如 `/data/my-new-folder`。

返回值

操作成功时不返回 `data`。

参数名 类型 描述
codenumber返回码，0表示成功
msgstring返回信息
datanull成功时为 null
请求示例 (cURL)
返回示例

请求示例 (cURL)

curl -X POST \
  http://127.0.0.1:6806/api/file/mkDir \
  -H 'Authorization: Token YOUR_API_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "path": "/data/my-new-folder/subfolder" 
}'

返回示例

{
    "code": 0,
    "msg": "",
    "data": null
}
