POST /api/file/readDirInfo

读取目录信息

源文件 (GitHub)
需要认证

接口描述

读取指定目录的详细信息，包括其自身属性和子项目（文件和目录）的列表。

注意： 此接口与 `/api/file/readDir` 类似，但返回的信息更侧重于目录本身的属性。

请求参数

参数名 类型 必选 描述
path string 是 要读取信息的目录路径，相对路径（相对于工作空间）。例如 `/data/templates`。

返回值

返回一个对象，包含目录的属性和子项目列表。

参数名 类型 描述
codenumber返回码，0表示成功
msgstring返回信息
dataobject目录信息对象。
data.namestring目录名。
data.pathstring目录的绝对路径。
data.bookmarkstring目录书签。
data.name1string目录别名。
data.iconstring目录图标 Emoji 字符。
data.memostring目录备注。
data.mtimenumber修改时间的 Unix 时间戳（毫秒）。
data.filesobject[]目录下的文件和子目录列表（结构同 `/api/file/readDir` 返回的 `data` 数组）。
请求示例 (cURL)
返回示例

请求示例 (cURL)

curl -X POST \
  http://127.0.0.1:6806/api/file/readDirInfo \
  -H 'Authorization: Token YOUR_API_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "path": "/data/templates"
}'
