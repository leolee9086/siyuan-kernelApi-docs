System (系统) API

首页

这里包含了与思源笔记系统层面功能相关的 API，例如工作空间管理、系统设置、版本信息等。

API 列表

  • /api/system/checkWorkspaceDir - 检查工作空间目录是否有效
  • /api/system/createWorkspaceDir - 创建工作空间目录并添加到列表
  • /api/system/exit - 退出程序 (高风险!)
  • /api/system/exportLog - 导出系统日志
  • /api/system/getMobileWorkspaces - 获取移动端工作空间列表 (移动端专用)
  • /api/system/getNetwork - 获取网络设置 (代理)
  • /api/system/getSysFonts - 获取系统字体列表
  • /api/system/removeWorkspaceDir - 从列表移除工作空间目录 (不删文件)
  • /api/system/removeWorkspaceDirPhysically - 物理删除工作空间目录 (极度危险!)
  • /api/system/setAppearanceMode - 设置外观模式（亮/暗）。
  • /api/system/setAutoLaunch - 设置是否开机自动启动应用。
  • /api/system/setDownloadInstallPkg - 设置是否自动下载并安装更新包。
  • /api/system/setFollowSystemLockScreen - 设置是否跟随系统锁屏。
  • /api/system/setGoogleAnalytics - 设置是否启用谷歌分析功能。
  • /api/system/setNetworkProxy - 设置网络代理配置。
  • /api/system/setNetworkServe - 设置是否在局域网中提供服务。
  • /api/system/setUILayout - 设置思源笔记的界面布局。
  • /api/system/setUploadErrLog - 设置是否允许上传错误日志。
  • /api/system/setWorkspaceDir - 设置要切换到的工作空间目录。
