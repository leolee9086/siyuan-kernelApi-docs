POST /api/history/getNotebookHistory

获取笔记本历史

需要认证

接口描述

获取指定笔记本的历史版本记录列表。这些历史记录是基于笔记本维度的快照，可以用于回滚整个笔记本到某个时间点。

请求参数

参数名 类型 必选 描述
notebook string 是 笔记本 ID
page number 否 页码，默认为 1
size number 否 每页数量，默认为 20

返回值

参数名 类型 描述
code number 返回码，0表示成功
msg string 返回信息
data object 包含历史记录列表和分页信息的对象
data.histories array 笔记本历史版本数组
data.histories[].path string 历史版本文件路径，用于回滚操作
data.histories[].created number 历史版本创建时间戳 (毫秒)
data.histories[].size number 历史版本文件大小 (字节)
data.count number 总历史记录数量
data.pageCount number 总页数
请求示例
返回示例

请求示例

POST /api/history/getNotebookHistory HTTP/1.1
Host: 127.0.0.1:6806
Content-Type: application/json
Authorization: Token your-token

{
  "notebook": "20200812220439-obsxfyn",
  "page": 1,
  "size": 10 
}
