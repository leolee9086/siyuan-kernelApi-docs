POST /api/history/getDocHistoryContent

获取文档历史内容

需要认证

接口描述

获取特定历史版本中某个文档的内容。此接口允许查看文档在历史版本中的完整内容，包括其中所有的块。

请求参数

参数名 类型 必选 描述
historyPath string 是 历史记录路径，通过getNotebookHistory接口获取
docID string 是 文档ID

返回值

参数名 类型 描述
code number 返回码，0表示成功
msg string 返回信息
data object 返回数据
data.id string 文档ID
data.rootID string 文档根ID
data.content string 文档内容
data.created string 创建时间
data.updated string 更新时间
data.box string 笔记本ID
data.path string 文档路径
data.hPath string 人类可读的文档路径
data.title string 文档标题
data.type string 文档类型
data.blocks array 文档中的块列表
data.blocks[].id string 块ID
data.blocks[].type string 块类型
data.blocks[].content string 块内容
data.blocks[].depth number 块深度
请求示例
返回示例

请求示例

POST /api/history/getDocHistoryContent HTTP/1.1
Host: 127.0.0.1:6806
Content-Type: application/json
Authorization: Token your-token

{
  "historyPath": "/data/history/20210808180117-czj9bvb/20230415120000",
  "docID": "20220301153724-vy37rik"
}
