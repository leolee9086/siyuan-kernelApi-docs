POST /api/history/rollbackDocHistory

回滚文档历史

需要认证 需要管理员权限
警告： 回滚操作不可逆，会覆盖当前文档内容。请在执行此操作前备份重要数据。

接口描述

将指定文档回滚到特定的历史版本。回滚操作会用历史版本的内容替换当前文档的内容，此操作不可逆，请谨慎使用。

请求参数

参数名 类型 必选 描述
historyPath string 是 历史记录路径，通过getNotebookHistory接口获取
docID string 是 需要回滚的文档ID

返回值

参数名 类型 描述
code number 返回码，0表示成功
msg string 返回信息
data null 无返回数据
请求示例
返回示例

请求示例

POST /api/history/rollbackDocHistory HTTP/1.1
Host: 127.0.0.1:6806
Content-Type: application/json
Authorization: Token your-token

{
  "historyPath": "/data/history/20210808180117-czj9bvb/20230415120000",
  "docID": "20220301153724-vy37rik"
}

返回示例

{
  "code": 0,
  "msg": "文档已回滚到指定的历史版本",
  "data": null
}
