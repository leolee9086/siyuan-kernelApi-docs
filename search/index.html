API/WS 列表

  • /api/search/findReplace - 在指定范围（整个工作空间、笔记本或文档）内查找文本，并将其替换为新的文本。
  • /api/search/fullTextSearchAssetContent - 对已建立索引的资源文件（Assets）内容进行全文搜索。
  • /api/search/fullTextSearchBlock - 全文搜索所有块内容，支持各种搜索条件和过滤选项。可以按关键词、查询语法、SQL或正则表达式进行搜索，并支持按创建时间、更新时间等排序。
  • /api/search/getAssetContent - 获取指定资源文件（Asset）的内容片段，并高亮指定的查询关键词。
  • /api/search/getEmbedBlock - 根据提供的块 ID 列表获取嵌入块内容。
  • /api/search/listInvalidBlockRefs - 列出工作空间中所有包含失效块引用的块。
  • /api/search/removeTemplate - 删除指定的模板片段文件。
  • /api/search/searchAsset - 根据关键词和文件扩展名搜索工作空间中的资源文件（Assets）。
  • /api/search/searchEmbedBlock - 执行嵌入块中的 SQL 查询语句，并返回匹配的块内容。
  • /api/search/searchRefBlock - 根据关键词搜索引用了特定块 ID 的块。
  • /api/search/searchTag - 在笔记中搜索标签，可根据关键词查找相关标签。
  • /api/search/searchTemplate - 根据关键词搜索工作空间中的模板片段。
  • /api/search/searchWidget - 根据关键词搜索工作空间中的挂件块 (widget)。
  • /api/search/updateEmbedBlock - 更新指定嵌入块 (query_embed 类型) 的 SQL 查询语句。
