POST /api/search/fullTextSearchDoc

全文搜索文档

需要认证

接口描述

在所有文档层级进行全文搜索，返回匹配的文档列表。此接口与全文搜索块不同，它是以文档为单位进行搜索和返回结果。

请求参数

参数名 类型 必选 描述
query string 是 搜索关键词
page number 否 页码，从1开始，默认为1
paths array 否 路径过滤，可以指定在哪些路径下搜索
method number 否 搜索方式，0: 关键字，1: 查询语法，2: SQL，3: 正则表达式，默认为0
orderBy number 否 排序方式，0: 相关度，1: 创建时间，2: 更新时间，3: 文档大小，默认为0

返回值

参数名 类型 描述
code number 返回码，0表示成功
msg string 返回信息
data object 返回数据
data.docs array 匹配到的文档列表
data.docs[].id string 文档ID
data.docs[].title string 文档标题
data.docs[].path string 文档路径
data.docs[].hPath string 人类可读的文档路径
data.docs[].created string 创建时间
data.docs[].updated string 更新时间
data.docs[].size number 文档大小（字节）
data.docs[].box string 所属笔记本ID
data.docs[].icon string 文档图标
data.docs[].content string 文档内容片段，包含搜索关键词的上下文
data.matchedDocCount number 匹配的文档总数
data.pageCount number 总页数
