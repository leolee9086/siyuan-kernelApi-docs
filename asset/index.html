API/WS 列表

  • /api/asset/fullReindexAssetContent - 对工作空间内的所有资源文件（assets 目录）执行全量的内容索引重建。这是一个耗时操作，通常在索引出现问题或需要强制更新时使用。
  • /api/asset/getDocAssets - 方法：POST 路径：/api/asset/getDocAssets
  • /api/asset/getDocImageAssets - 获取指定文档块直接引用的所有图片类型的资源文件路径列表。
  • /api/asset/getFileAnnotation - 获取指定资源文件的标注信息 (PDF 文件等)。
  • /api/asset/getImageOCRText - 获取指定图片资源文件中包含的 OCR 文本（如果存在）。
  • /api/asset/getMissingAssets - 获取工作空间中被引用但实际文件已丢失的资源列表。
  • /api/asset/getUnusedAssets - /api/asset/getUnusedAssets 的接口描述待补充。
  • /api/asset/insertLocalAssets - 将本地文件系统中的一个或多个文件复制到思源笔记的 assets 目录，并返回它们在笔记中可用的资源链接。
  • /api/asset/ocr - 对指定的图片资源文件执行 OCR（光学字符识别），并将识别出的文本保存起来，以便后续搜索和获取。
  • /api/asset/removeUnusedAsset - 删除一个指定的、未被任何文档引用的资源文件。
  • /api/asset/removeUnusedAssets - 删除工作空间中所有未被任何文档引用的资源文件。这是一个危险操作，请谨慎使用。
  • /api/asset/renameAsset - 重命名一个指定的资源文件。同时会自动更新所有引用该资源的地方。
  • /api/asset/resolveAssetPath - 将相对于 assets 目录的资源路径解析为其在当前操作系统上的绝对路径。
  • /api/asset/setFileAnnotation - 设置指定资源文件的标注信息 (通常用于 PDF 文件)。
  • /api/asset/setImageOCRText - 设置指定图片资源文件的 OCR 文本内容。这通常用于手动修正或添加 OCR 结果。
  • /api/asset/statAsset - 获取指定资源文件的统计信息，如大小、修改时间等。
  • /api/asset/upload - 上传一个或多个文件到指定的文档块关联的 assets 目录。通常用于通过 HTTP 表单上传文件。
  • /api/asset/uploadCloud - 将本地资源文件上传到配置的云存储（如 S3）。通常用于云端同步相关的操作。
