POST /api/lute/markdown2HTML

Markdown转HTML

需要认证

接口描述

将输入的Markdown文本转换为HTML格式。使用思源笔记内置的Lute引擎进行转换，确保与笔记内部渲染效果一致。

请求参数

参数名 类型 必选 描述
markdown string 是 要转换的Markdown文本
options object 否 Lute转换选项（具体选项待确认，可能包括GFM、脚注等支持开关）

返回值

参数名 类型 描述
code number 返回码，0表示成功
msg string 返回信息
data string 转换后的HTML文本
请求示例
返回示例

请求示例

POST /api/lute/markdown2HTML HTTP/1.1
Host: 127.0.0.1:6806
Content-Type: application/json
Authorization: Token your-token

{
  "markdown": "# 标题1\n\n这是**加粗**文本。\n\n- 列表项1\n- 列表项2"
}

返回示例

{
  "code": 0,
  "msg": "",
  "data": "<h1>标题1</h1>\n<p>这是<strong>加粗</strong>文本。</p>\n<ul>\n<li>列表项1</li>\n<li>列表项2</li>\n</ul>"
}
