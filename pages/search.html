搜索 API

搜索相关的API接口，用于在工作空间内搜索内容、标签、模板、资源文件等，并执行查找替换操作。

  • 搜索标签 /api/search/searchTag

    根据关键词搜索工作空间中的标签

  • 搜索模板 /api/search/searchTemplate

    根据关键词搜索工作空间中的模板片段

  • 全文搜索块 /api/search/fullTextSearchBlock

    根据条件在所有块中进行全文搜索

  • 查找替换 /api/search/findReplace

    在指定范围内查找并替换文本
