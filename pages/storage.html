存储 API

存储相关的API接口，用于管理本地存储和数据持久化。

  • 设置本地存储 /api/storage/setLocalStorage

    设置本地存储数据

  • 获取本地存储 /api/storage/getLocalStorage

    获取本地存储数据

  • 设置本地存储值 /api/storage/setLocalStorageVal

    设置本地存储中的特定值

  • 删除本地存储值 /api/storage/removeLocalStorageVals

    删除本地存储中的特定值

  • 设置条件 /api/storage/setCriterion

    设置查询条件

  • 获取条件 /api/storage/getCriteria

    获取查询条件

  • 删除条件 /api/storage/removeCriterion

    删除查询条件

  • 获取最近文档 /api/storage/getRecentDocs

    获取最近访问的文档
