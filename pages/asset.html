资源 Asset API

用于管理资源文件（图片、附件等）的 API 接口。

  • 全量重建资源内容索引 /api/asset/fullReindexAssetContent

    重新为所有资源文件建立内容索引，用于搜索。

  • 获取文档图片资源 /api/asset/getDocImageAssets

    获取指定文档块引用的所有图片资源。

  • 获取文件标注 /api/asset/getFileAnnotation

    获取指定资源文件的标注信息。

  • 获取图片 OCR 文本 /api/asset/getImageOCRText

    获取指定图片资源文件中的 OCR 文本。

  • 获取丢失资源 /api/asset/getMissingAssets

    获取被引用但文件已丢失的资源列表。

  • 获取未使用资源 /api/asset/getUnusedAssets

    获取存在但未被引用的资源文件列表。

  • 插入本地资源 /api/asset/insertLocalAssets

    将本地文件复制到 assets 目录并返回链接。

  • OCR 识别 /api/asset/ocr

    对指定图片资源执行 OCR 并保存结果。

  • 删除单个未使用资源 /api/asset/removeUnusedAsset

    删除一个指定的、未被引用的资源文件。

  • 删除所有未使用资源 /api/asset/removeUnusedAssets

    删除所有未被引用的资源文件。

  • 重命名资源 /api/asset/renameAsset

    重命名一个指定的资源文件。

  • 解析资源路径 /api/asset/resolveAssetPath

    将相对资源路径解析为绝对路径。

  • 设置文件标注 /api/asset/setFileAnnotation

    设置指定资源文件的标注信息。

  • 设置图片 OCR 文本 /api/asset/setImageOCRText

    设置指定图片资源文件的 OCR 文本。

  • 获取资源统计信息 /api/asset/statAsset

    获取指定资源文件的统计信息。

  • 上传资源 /api/asset/upload

    上传一个或多个文件到资源目录。

  • 上传到云端 /api/asset/uploadCloud

    将本地资源文件上传到配置的云存储。
