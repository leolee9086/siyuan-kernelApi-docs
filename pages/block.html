块 API

块相关的API接口，用于创建、查询、修改和删除内容块。

  • 获取块信息 /api/block/getBlockInfo

    获取指定块的详细信息

  • 获取块DOM /api/block/getBlockDOM

    获取指定块的DOM结构

  • 获取块Kramdown /api/block/getBlockKramdown

    获取指定块的Kramdown源码

  • 获取子块 /api/block/getChildBlocks

    获取指定块的所有子块

  • 插入块 /api/block/insertBlock

    在指定位置插入新块

  • 前置插入块 /api/block/prependBlock

    在指定块的开头插入子块

  • 后置插入块 /api/block/appendBlock

    在指定块的末尾插入子块

  • 更新块 /api/block/updateBlock

    更新指定块的内容

  • 删除块 /api/block/deleteBlock

    删除指定的块

  • 移动块 /api/block/moveBlock

    移动块到新的位置

  • 折叠块 /api/block/foldBlock

    折叠指定的块

  • 展开块 /api/block/unfoldBlock

    展开指定的块

  • 获取块面包屑 /api/block/getBlockBreadcrumb

    获取块的层级路径面包屑

  • 获取引用ID /api/block/getRefIDs

    获取指定块的所有引用ID

  • 转移块引用 /api/block/transferBlockRef

    将一个块的引用转移到另一个块

  • 检查块是否存在 /api/block/checkBlockExist

    检查指定的块是否存在于数据库中

  • 检查块是否折叠 /api/block/checkBlockFold

    检查指定的块是否被折叠及其是否为根块

  • 通过引用文本获取定义块 ID /api/block/getBlockDefIDsByRefText

    根据引用文本查找对应的定义块 ID

  • 获取块索引 /api/block/getBlockIndex

    获取块在其父块下的位置索引

  • 获取块兄弟节点 ID /api/block/getBlockSiblingID

    获取块的父块、上一个和下一个兄弟块 ID

  • 获取块树信息 /api/block/getBlockTreeInfos

    批量获取块的基本树状信息

  • 批量获取块索引 /api/block/getBlocksIndexes

    批量获取块在其父块下的位置索引

  • 批量获取块字数 /api/block/getBlocksWordCount

    计算并返回一组块的总字数（含子块）

  • 获取内容块字数 /api/block/getContentWordCount

    计算并返回指定内容块的字数（不含子块）

  • 获取块 DOM 纯文本 /api/block/getDOMText

    获取块渲染后的 DOM 中的纯文本内容

  • 获取文档信息 /api/block/getDocInfo

    获取指定块所属文档的基本信息

  • 获取标题删除事务 /api/block/getHeadingDeleteTransaction

    获取删除标题块所需的事务操作

  • 获取标题级别调整事务 /api/block/getHeadingLevelTransaction

    获取调整标题级别所需的事务操作

  • 设置块提醒 /api/block/setBlockReminder

    为块设置提醒时间

  • 交换块引用与定义 /api/block/swapBlockRef

    交换块引用和其定义块的内容
