系统 API

系统相关的API接口，用于获取系统信息、设置系统配置等。

  • 获取启动进度 /api/system/bootProgress

    获取思源笔记的启动进度信息

  • 获取版本 /api/system/version

    获取思源笔记的版本信息

  • 获取当前时间 /api/system/currentTime

    获取思源笔记服务器的当前时间

  • 添加UI进程 /api/system/uiproc

    添加UI进程信息

  • 登录鉴权 /api/system/loginAuth

    进行登录鉴权

  • 登出鉴权 /api/system/logoutAuth

    进行登出鉴权

  • 获取验证码 /api/system/getCaptcha

    获取验证码图片

  • 获取表情配置 /api/system/getEmojiConf

    获取思源笔记的表情配置

  • 设置API令牌 /api/system/setAPIToken

    设置思源笔记的API访问令牌

  • 设置访问授权码 /api/system/setAccessAuthCode

    设置思源笔记的访问授权码

  • 获取配置 /api/system/getConf

    获取思源笔记的配置信息

  • 获取工作空间 /api/system/getWorkspaces

    获取工作空间列表

  • 获取更新日志 /api/system/getChangelog

    获取思源笔记的更新日志
