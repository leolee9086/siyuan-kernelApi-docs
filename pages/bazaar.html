集市 (Bazaar) API

用于管理思源笔记的扩展，包括插件、主题、图标、挂件和模板等。

获取列表

  • 获取集市插件列表 /api/bazaar/getBazaarPlugin

    获取官方集市中可用的插件列表，支持关键词搜索。

  • 获取已安装插件列表 /api/bazaar/getInstalledPlugin

    获取本地已安装的插件列表，支持关键词搜索。

  • 获取集市挂件列表 /api/bazaar/getBazaarWidget

    获取官方集市中可用的挂件列表，支持关键词搜索。

  • 获取已安装挂件列表 /api/bazaar/getInstalledWidget

    获取本地已安装的挂件列表，支持关键词搜索。

  • 获取集市图标列表 /api/bazaar/getBazaarIcon

    获取官方集市中可用的图标主题列表，支持关键词搜索。

  • 获取已安装图标列表 /api/bazaar/getInstalledIcon

    获取本地已安装的图标主题列表，支持关键词搜索。

  • 获取集市模板列表 /api/bazaar/getBazaarTemplate

    获取官方集市中可用的模板列表，支持关键词搜索。

  • 获取已安装模板列表 /api/bazaar/getInstalledTemplate

    获取本地已安装的模板列表，支持关键词搜索。

  • 获取集市主题列表 /api/bazaar/getBazaarTheme

    获取官方集市中可用的主题列表，支持关键词搜索。

  • 获取已安装主题列表 /api/bazaar/getInstalledTheme

    获取本地已安装的主题列表，支持关键词搜索。

  • 获取可更新包列表 /api/bazaar/getUpdatedPackage

    检查本地已安装的扩展包（插件、主题等），返回有可用更新的列表。

安装扩展

  • 安装插件 /api/bazaar/installBazaarPlugin

    从集市下载并安装指定的插件。

  • 安装挂件 /api/bazaar/installBazaarWidget

    从集市下载并安装指定的挂件。

  • 安装图标 /api/bazaar/installBazaarIcon

    从集市下载并安装指定的图标主题。

  • 安装模板 /api/bazaar/installBazaarTemplate

    从集市下载并安装指定的模板。

  • 安装主题 /api/bazaar/installBazaarTheme

    从集市下载并安装指定的主题。

卸载扩展

  • 卸载插件 /api/bazaar/uninstallBazaarPlugin

    卸载本地已安装的指定插件。

  • 卸载挂件 /api/bazaar/uninstallBazaarWidget

    卸载本地已安装的指定挂件。

  • 卸载图标 /api/bazaar/uninstallBazaarIcon

    卸载本地已安装的指定图标主题。

  • 卸载模板 /api/bazaar/uninstallBazaarTemplate

    卸载本地已安装的指定模板。

  • 卸载主题 /api/bazaar/uninstallBazaarTheme

    卸载本地已安装的指定主题。

其他操作

  • 获取包 README /api/bazaar/getBazaarPackageREAME

    获取指定扩展包（插件、主题等）的 README 文件内容（通常是渲染后的 HTML）。

  • 批量更新包信息 /api/bazaar/batchUpdatePackage

    强制检查并更新本地已安装扩展包的版本等信息。
