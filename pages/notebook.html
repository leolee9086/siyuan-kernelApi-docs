笔记本 API

笔记本相关的API接口，用于管理笔记本、设置笔记本配置等。

  • 列出笔记本 /api/notebook/lsNotebooks

    获取所有笔记本的列表

  • 打开笔记本 /api/notebook/openNotebook

    打开一个指定的笔记本

  • 关闭笔记本 /api/notebook/closeNotebook

    关闭一个指定的笔记本

  • 重命名笔记本 /api/notebook/renameNotebook

    重命名一个指定的笔记本

  • 创建笔记本 /api/notebook/createNotebook

    创建一个新的笔记本

  • 删除笔记本 /api/notebook/removeNotebook

    删除一个指定的笔记本

  • 获取笔记本配置 /api/notebook/getNotebookConf

    获取笔记本的配置信息

  • 设置笔记本配置 /api/notebook/setNotebookConf

    设置笔记本的配置信息

  • 调整笔记本排序 /api/notebook/changeSortNotebook

    调整笔记本的排序

  • 设置笔记本图标 /api/notebook/setNotebookIcon

    设置笔记本的图标
