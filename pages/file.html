文件 API

文件操作相关的API接口，用于管理思源笔记工作空间中的文件和资源。

  • 获取文件 /api/file/getFile

    获取指定路径的文件内容

  • 保存文件 /api/file/putFile

    保存内容到指定路径的文件

  • 删除文件 /api/file/removeFile

    删除指定路径的文件

  • 读取目录 /api/file/readDir

    读取指定目录下的文件和子目录

  • 读取目录信息 /api/file/readDirInfo

    读取指定目录的详细信息

  • 创建目录 /api/file/mkDir

    创建指定路径的目录

  • 删除目录 /api/file/removeDir

    删除指定路径的目录及其内容

  • 重命名文件 /api/file/renameFile

    重命名指定路径的文件

  • 复制文件 /api/file/copyFile

    复制指定路径的文件或目录

  • 获取唯一文件名 /api/file/getUniqueFilename

    在指定目录下生成唯一的文件名

  • 全局复制文件 /api/file/globalCopyFiles

    从外部复制文件到工作空间
