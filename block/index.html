Block (块) API

首页

这里包含了与块（Block）操作相关的 API，块是思源笔记内容的基本单位。

API 列表

  • /api/block/appendBlock - 在父块末尾追加块
  • /api/block/batchUpdateBlock - 批量更新块内容
  • /api/block/deleteBlock - 删除块
  • /api/block/foldBlock - 折叠块
  • /api/block/getBlockBreadcrumb - 获取块的面包屑路径
  • /api/block/getBlockDefIDsByRefText - 通过引用文本获取定义块ID列表
  • /api/block/getBlockDOM - 获取块的 DOM 结构
  • /api/block/getBlockIndex - 获取块的索引信息（已废弃？）
  • /api/block/getBlockInfo - 获取块的基本信息（所属笔记本、路径等）
  • /api/block/getBlockKramdown - 获取块的 Kramdown 源码
  • /api/block/getBlockRefIDs - 获取块引用的 ID 列表
  • /api/block/getBlockRefIDsByFileAnnotationID - 通过文件标注 ID 获取块引用 ID 列表
  • /api/block/getBlocksIndexes - 批量获取块的索引信息（已废弃？）
  • /api/block/getChildBlocks - 获取子块列表
  • /api/block/getRefIDs - 获取引用了指定块的块 ID 列表（反向链接）
  • /api/block/getTailChildBlocks - 获取末尾 N 个子块
  • /api/block/insertBlock - 在指定位置插入块
  • /api/block/moveBlock - 移动块
  • /api/block/prependBlock - 在父块开头插入块
  • /api/block/transferBlockRef - 转移块引用关系
  • /api/block/unfoldBlock - 展开块
  • /api/block/updateBlock - 更新单个块内容
  • /api/block/getDocsInfo - 获取多个文档块信息
  • /api/block/getBlockDef - 获取块定义
  • /api/block/getBlockMarkdownAndHTML - 获取块 Markdown 和 HTML
  • /api/block/getDocInfo - 获取文档块信息
  • /api/block/getHeadingLevelTransaction - 获取设置标题层级事务
  • /api/block/getRefIDsByFileAnnotationID - 根据文件标注 ID 获取引用块 ID
  • /api/block/getRefText - 获取引用文本
  • /api/block/getTreeStat - 获取树状态
  • /api/block/setBlockAttrs - 设置块属性
  • /api/block/sql - 执行 SQL 查询
  • /api/block/checkBlockExist - 检查块是否存在
  • /api/block/checkBlocksExist - 检查块是否存在（批量）
  • /api/block/findBlock - 查找块
  • /api/block/getBlockByID - 根据 ID 获取块
  • /api/block/getBlockAttrs - 获取块属性
  • /api/block/getBlocksInfo - 获取多个块信息
  • /api/block/getDocInfoByBlockID - 根据块 ID 获取文档信息
  • /api/block/getFiles - 获取文件列表
  • /api/block/getHeadingLevel - 获取标题层级
  • /api/block/setBlockRef - 设置块引用
  • /api/block/updateBlockBatch - 批量更新块（旧，建议使用 batchUpdateBlock）
  • /api/block/createDocWithMd - 使用 Markdown 创建文档
  • /api/block/appendBlockBatch - 批量追加块
  • /api/block/prependBlockBatch - 批量前置块
  • /api/block/insertBlockBatch - 批量插入块
  • /api/block/updateBlocks - 更新多个块
  • /api/block/deleteBlocks - 批量删除块
  • /api/block/getBlockChildBlocks - 获取块的子块
  • /api/block/getDocOutline - 获取文档大纲
  • /api/block/getRecentUpdatedBlocks - 获取最近更新的块
