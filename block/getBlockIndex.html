获取块索引

在 GitHub 上查看源码

POST /api/block/getBlockIndex

描述

获取指定块在其父块（或文档根）下的子块列表中的位置索引（从 0 开始）。

请求参数

请求体需要是 JSON 格式。

参数名 类型 描述 必需
id string 要获取索引的块 ID。 是

响应结果

响应体是 JSON 格式。code 为 0 表示成功。data 包含块的索引值。


{
  "code": 0,
  "msg": "",
  "data": 2 // 例如，表示该块是其父块下的第 3 个子块
}
