删除块 API (`/api/block/deleteBlock`)

在 GitHub 上查看源码 返回块 API 列表 返回 API 主页

接口描述

该接口用于删除指定 ID 的块。如果删除的是文档块，其关联的文件也会被放入回收站。

请求

方法: POST

路径: `/api/block/deleteBlock`

认证: 需要 Token

请求体 (JSON)

参数 类型 必需 描述
`id` string 是 要删除的块的 ID。

请求示例

{
    "id": "20230517100000-abcdefgh"
}

响应

成功响应 (200 OK)

成功时，返回一个包含执行的操作事务信息的对象。

{
    "code": 0,
    "msg": "",
    "data": {
        "transactions": [
            {
                "doOperations": [
                    {
                        "action": "delete",
                        "id": "20230517100000-abcdefgh"
                        // 其他操作相关的字段...
                    }
                ],
                // 其他事务相关的字段...
            }
        ]
    }
}

失败响应

如果请求失败（例如，ID 无效或块不存在），将返回错误信息。

{
    "code": -1,
    "msg": "Invalid block ID pattern [无效的ID]", // 或 "Block not found [id=...]"
    "data": null
}
