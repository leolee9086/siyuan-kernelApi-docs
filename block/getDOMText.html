获取块 DOM 纯文本

在 GitHub 上查看源码

POST /api/block/getDOMText

描述

获取指定块渲染后的 DOM 结构中的纯文本内容，移除了所有 HTML 标签。

请求参数

请求体需要是 JSON 格式。

参数名 类型 描述 必需
id string 目标块 ID。 是

响应结果

响应体是 JSON 格式。code 为 0 表示成功。data 包含提取出的纯文本字符串。


{
  "code": 0,
  "msg": "",
  "data": "这是块的纯文本内容。"
}
