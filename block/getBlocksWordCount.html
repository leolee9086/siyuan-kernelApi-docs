批量获取块字数

在 GitHub 上查看源码

POST /api/block/getBlocksWordCount

描述

计算并返回一组块 ID 对应的总字数（包括所有子块）。

请求参数

请求体需要是 JSON 格式。

参数名 类型 描述 必需
ids [string] 要计算字数的块 ID 数组。 是

响应结果

响应体是 JSON 格式。code 为 0 表示成功。data 包含计算出的总字数。


{
  "code": 0,
  "msg": "",
  "data": {
    "count": 1234 // 这些块的总字数
  }
}

注意：这是一个社区维护的文档，可能与官方最新版本存在差异。
